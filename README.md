# PokéNotes

Aplicação de notas e agenda com tema retro de Pokémon.

## Como usar

Primeiro, instale as dependências:

```bash
npm install
```

Depois, rode em modo de desenvolvimento:

```bash
npm run dev
```

Acesse http://localhost:3000 no seu navegador.

## Build para produção

Para gerar uma versão otimizada:

```bash
npm run build
npm start
```

## Tecnologias usadas

- Next.js 14
- TypeScript
- Tailwind CSS
- Zustand
- Framer Motion
- Markdown Support
